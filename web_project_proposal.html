Web Project Proposal

Subject/purpose

This site would be for a fishing tournament in Texas. It would contain the rules and regulations, maps and locations for each fishing site, a gallery of pictures of the winners, etc. Basically the things required to run a tournament and to let the people know what is going on and when.

Pages

  1. Home/About us: This page includes the basic "about us" information, as well as any important last-minute announcements, contact information, etc.
  2. Rules: The rules page. This would include all of the official rules of the tournament as outlined by the leaders/creators of the tournament.
  3. Registration: This page would have either a registration form to print or an actual submission form that is filled out on the page and then sent to wherever for review.
  4. Gallery: The gallery page would feature pictures of the tournament in general as well as photos of each session's winners.
  5. Schedule: The schedule page would have a schedule lined out as well as maps to each tournament location.
  6. Results: The results page would simply have all of the results from each tournament session, as well as the master weigh-in sheet, updated for each session.
